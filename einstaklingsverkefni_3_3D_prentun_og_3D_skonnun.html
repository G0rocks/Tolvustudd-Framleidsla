Project 3 - Additive manufacturing

This project is all about designing and 3D printing something that you can not manufacture using a 3-axis milling machine and scanning an object in 3D.

Part 0 - Test

Before starting we are supposed to to a test to see the capabilities of the printer. The test can be done in 3 person groups and is documented here.

Part 1 - Idea

Firstly, for the 3D printing project we need to know what we're designing, we need an idea. After thinking for a while and cleaning the house and thinking some more, we get the idea to make a broom handle for the broom at home, see image below. You see, the thing is, ever since we got this broom the blue handle at the top has been slowly falling off and last year I taped it to the broomstick but the tape fell off too not long ago so we need this fixed. We need:

  • A way to make the handle stay put without damaging/removing material (this is additive manufacturing)
  • The handle to be strong enough to handle (heh, see what I did there) the load
  • A good grip for the hand
  • To be able to 3D print our solution
  • Bonus: If we can make the handle comfortable

Not thinking about the design requirements, our first idea is to drill a hole through the current handle and the broomstick, put a bolt through there and a nut on the other side, tighten and done. But that won't work, since it's not additive. Our second idea, this time considering the requirements, uses a clamping mechanism, similar to the one you see on hand brakes on bikes, see image below, this clamping mechanism will attach to the broomstick.

After getting this idea approved by the professor, we move on to the next step og the project.
