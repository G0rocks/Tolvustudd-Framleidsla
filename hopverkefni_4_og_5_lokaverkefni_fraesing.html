Project 4 and 5 - Group project - Milling

This project is about using a CNC milling machine to either make a wax mold or some larger wood project and presenting our project at the end. This is a group project and me and Björn Virgill are doing this project together. While doing this project we need to implement project organizational practices and document the project management as well as the project design and manufacturing. Both me and Björn Virgill documented this page together but Björn Virgill documented some things here and I documented some things here.

Part 1 - Projet management

Both me and Björn Virgill are used to kanban style management, especially for a short-term project like this one so we opted for using a website called trello after we had sketched out our initial plan.

We were really excited about it!

We had even made sketches for the bike light attachment we were going to cast and the TV cabinet we were going to mill

And then we realized we read the project details wrong, we were only supposed to either cast something or mill something, so we decided on milling something and made an actual plan:

Some milestones were decided on in the beginning, having the design ready, manufacturing, assembly and presentation. The design is a collaborative project in another course as well which needs to use the finished project so the schedule for design, manufacturing and assembly was set tight. Having a tight schedule is also practical since Björn Virgill is going abroad soon so both of us want to finish the project sooner rather than later.
