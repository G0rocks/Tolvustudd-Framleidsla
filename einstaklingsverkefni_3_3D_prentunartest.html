3D printer test

This is a part of project 3 which is all about designing and 3D printing something that you can not manufacture using a 3-axis milling machine and scanning an object in 3D. A part of that project is

Part 1 - Preparation

To prepare we search the web for a way to test the capabilities of the 3D printer, stumbling upon this all in one 3D printer test it is decided that is what we shall use. Oh and btw the printer being used is a monoprice select mini v2.

Part 2 - Implementation

Settings

After downloading the file we want to print it and to do that we must configure some settings in ultimaker cura, the slicer being used, because this is what the file looks like after just uploading it:

The settings we must configure are various, such as setting the build plate adhesion to a skirt so we get a nice flow of filament going on before starting the actual print, this helps make sure the first layer is printed well.

Setting the infill to a 20% grid because that's a common infill for many prints, reduces the time need for printing from the original 70% infill file we had and saves filament.

Setting the profile to 0.175mm since that is a common profile to use and we want to get resulst for settings which are often used.

Setting the supports to none since we want to see how much of an overhang the printer can handle among other things like bridging.

Setting the wall line count to 4 because even though we want standard results we're still a little scared the overhang will fall over.

Then all that was left to do was to upload and wait. The estimated printing time was 7 hours and 30 minutes and the estimated material use was 48g or 15.98m of filament.
